Verification of test comment

[[ variable + option ]]

-n Variable is not null
-z Variable is null
-d  Verify if it is a directory or if the directory exists
-e  Verify if the file exists
-r  Verify if the file has read permission
-w  Verify if the file has write permission
-x  Verify if the file has execute permission
-L  Verify if it is a symbolic link
-O  Verify if the logged-in user is the owner
-f  Verify if it is a regular file
-ot Verify archive is OLDER THAN
-nt Verify archive is NEWER THAN
